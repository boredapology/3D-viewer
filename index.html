생성형 AI 자동차 3D 생성 평가 모델 뷰어
Hongik iiD-Lab

안녕하세요. 전문가님

본 뷰어에는 이미지 생성형 AI와 3D 생성형 AI를 결합한 워크플로우를 통해 생성된 자동차 3D 모델들이 제시되어 있습니다.

평가 대상은 3D 생성 도구 자체의 성능 비교가 아니라, 차량 이미지를 생성·수정한 뒤 이를 3D 모델로 연결하는 방식이 자동차 디자인 실무에서 얼마나 유효한지를 확인하는 것입니다.

상단 메뉴에서 싱글뷰 이미지(Baseline), 멀티뷰(Proposed), 파츠 변형(Application) 결과를 순서대로 확인해 주세요.

  • 싱글뷰 이미지: 단일 시점 차량 이미지를 기반으로 생성된 3D 모델
  • 멀티뷰: 전·측·후면 정보를 포함한 입력을 통해 시점 일관성을 높이도록 생성한 3D 모델
  • 파츠 변형: 차량 이미지를 수정하거나 특정 스타일을 반영하여 생성한 3D 모델

각 모델을 확인하시면서, 차량 디자인의 일관성, 후면/측면 조형 반영, 파츠 변화 반영, 차체 정체성 유지, 비례·볼륨 검토 가능성을 중심으로 평가해 주시기 바랍니다.

자동차 디자인을 위한
AI 기반 이미지 생성 및 3D 생성 워크플로우

1. 자동차 이미지 생성

  • 싱글뷰 차량 이미지 생성
  • 멀티뷰 차량 이미지 구성

2. 스타일 변형 및 디자인 수정

  • 파츠 단위 수정
  • 인페인팅 / LoRA 기반 스타일 반영

3. 이미지를 기반으로 3D 모델 생성

  • Image-to-3D 기반 Mesh 생성
  • 조형 검토 및 결과 비교
차량 이미지를 생성하고, 필요한 경우 디자인을 수정하거나 학습된 스타일을 반영한 뒤, 이를 바탕으로 3D 메쉬 모델을 생성하는 워크플로우입니다.
