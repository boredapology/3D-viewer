Hongik iiD-Lab

안녕하세요. 전문가님

본 뷰어는 생성형 AI 기반 자동차 디자인 프로세스에서, 단일 시점 입력의 한계를 극복하기 위한 '입력 이미지 설계·통제 전략'의 유효성을 평가하기 위한 연구 도구입니다.

본 연구는 3D 생성 모델의 절대적인 메쉬(Mesh) 품질을 비교하는 것이 아닙니다. 멀티뷰 구성을 통한 비가시 영역(후면)의 조형 반영과 인페인팅 기반 파츠 단위 변형의 통제 가능성이 실제 자동차 초기 디자인 발산 및 3D 검증 단계에서 실무적으로 유의미한지 워크플로우 관점에서 평가해 주시기 바랍니다.

상단 메뉴에서 싱글뷰 이미지(Baseline), 멀티뷰(Proposed), 파츠 변형(Application) 결과를 순서대로 확인해 주세요.

  • 싱글뷰 이미지 (기준 모델): 측·후면 조형 정보가 부족할 때, 3D 생성에서 나타나는 형태 생략이나 구조적 한계 확인
  • 멀티뷰 (다중 시점 전략): 후면 디자인 정보의 명확한 반영과, 전·측·후 시점 간 조형 일관성 유지 여부 확인
  • 파츠 변형 (인페인팅 전개 전략): 차선의 기본 비례와 정체성을 유지한 채, 타겟 파츠의 디자인 변화만 3D에 수평 전개되는지 확인

자동차 디자인을 위한
AI 기반 이미지 생성 및 3D 생성 워크플로우

1. 자동차 이미지 생성

  • 싱글뷰 차량 이미지 생성
  • 멀티뷰 차량 이미지 구성

2. 스타일 변형 및 디자인 수정

  • 파츠 단위 수정
  • 인페인팅 / LoRA 기반 스타일 반영

3. 이미지를 기반으로 3D 모델 생성

  • Image-to-3D 기반 Mesh 생성
  • 조형 검토 및 결과 비교
차량 이미지를 생성하고, 필요한 경우 디자인을 수정하거나 학습된 스타일을 반영한 뒤, 이를 바탕으로 3D 메쉬 모델을 생성하는 워크플로우입니다.
